### 풀이
  - 공식이 필요한 문제
    - 대각선을 지나는 사각형 개수 = w + h - gcd(w,h)
  
  - gcd(x,y)
    - 큰 수에서 작은 수를 제수과 나머지로 계속해서 나눴을 때, 나머지가 0이 될 때의 제수
  
```javascript
function solution(w, h) {
    function gcd(x,y){
        return y==0 ? x : gcd(y, x%y)
    }
    var answer = w*h - (w+h-gcd(w,h))
    return answer;
}
```
