
# 점프와 순간이동
  - [점프와 순간이동](https://programmers.co.kr/learn/courses/30/lessons/12980#)

### solution
  - n이 짝수이면, 홀수가 나올 때까지 /2
  - n이 홀수이면, n-1의 최솟값+1
  - 1일 때, 최솟값 1

  ```javascript
  function solution(n){
    const hash = {1:1};
    function getNum(n){
        if(!hash[n]) hash[n] = n % 2 == 0 ? getNum(n/2) : getNum(n-1)+1;
        return hash[n]
    }
    return getNum(n);
  }
  ```
