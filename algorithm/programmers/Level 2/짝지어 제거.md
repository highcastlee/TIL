### 풀이
  - 스택에 넣으면서 마지막 요소랑 겹치면 연속이므로 요소 제거
  - 안 겹치면 스택에 추가

```javascript
function solution(s){
    let stack = [s[0]]
    for (let i=1; i<s.length; i++){
        if(stack[stack.length-1] == s[i]) stack.pop()
        else stack.push(s[i])
    }
    if(stack.length > 0) return 0
    else return 1
}
```
