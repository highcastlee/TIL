
# 점화식 (= 재귀식)
  - 수열에서 이웃하는 두 개의 항 사이에 성립하는 관계를 나타낸 관계식
  - 대표적 점화식
    - 등차 수열 : f(n) = f(n-1) + a
    - 등비 수열 : f(n) = f(n-1) * a
    - 팩토리얼 : f(n) = f(n-1) * n
    - 피보나치 수열 : f(n) = f(n-1) + f(n-2)
    

  ```javascript
  let result;

  function recursive(n){
    if (n === 1){
      return 1;
    }
    return n + recursive(n-1);
  }

  result = recursice(3);
  console.log(result);    // 6   (3 + 2 + 1)
  ```
